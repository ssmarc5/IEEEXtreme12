    Barrett Reduction

    Time limit: 500 ms
    Memory limit: 256 MB

    The C++\text{C++}C++ compiler optimizes the division by a constant DDD by multiplying your number by a constant AAA and then shifting the result BBB bits. This process is called Barrett reduction.

    In this challenge we'll consider the division of an unsigned 32 bit integer by another unsigned 32 bit integer.

    Given AAA and BBB, your task is to find the minimum possible value of DDD, for which the division is correct.

    Note: it's guaranteed that for the given values AAA and BBB there exists an integer DDD such that for every number 0≤X<2310 \leq X < 2^{31}0≤X<2​31​​, ⌊XD⌋=⌊X×A2B⌋\lfloor \frac{X}{D} \rfloor = \lfloor \frac{X \times A}{2^{B}} \rfloor⌊​D​​X​​⌋=⌊​2​B​​​​X×A​​⌋, where ⌊X⌋\lfloor X \rfloor⌊X⌋ represents the floor (integer part) of the result.

    E.g: ⌊0.4⌋=0\lfloor 0.4 \rfloor = 0⌊0.4⌋=0, ⌊6.8⌋=6\lfloor 6.8 \rfloor = 6⌊6.8⌋=6.

    Note that the challenge only uses unsigned integers, so there are no problems regarding using floor on negative numbers.

    Standard input

    The first line contains two integers AAA and BBB.

    Standard output

    The first line should contain one integer DDD.

    Constraints and notes

    • 0≤A<2320 \leq A < 2^{32}0≤A<2​32​​
    • 0≤B≤630 \leq B \leq 630≤B≤63
    • 1≤D<2311 \leq D < 2^{31}1≤D<2​31​​
